
# CreatingTool - Telegram Automation Tool

**Version:** 2.3 
**Developer:** Dark Storm

---

## Description
CreatingTool is a Python-based Telegram automation tool that allows you to create supergroups, send messages automatically, and manage multiple accounts efficiently. The tool is designed to help you automate repetitive tasks safely while handling account limits and flood waits.

---

## Features
- Add, delete, and list Telegram accounts easily.
- Create up to 50 supergroups per account per run.
- Automatically send messages to groups.
- Multi-thread support: run on one or all accounts simultaneously.
- Automatic update system from GitHub releases.
- Handles FloodWait errors gracefully.
- Countdown timers to avoid Telegram bans.

---

## Usage :
1. Run the script:
   ```bash
   python3 Main.py

----
## Menu options:
1. Add account – add a new Telegram account.how accounts – list all accounts.
2. Run on one – run the tool on a single account.
3. Run on all – run the tool on all accounts.
4. Delete account – remove an account safely.
5. Exit – close the tool.

### Notes
Run the script once per day to reduce the risk of Telegram bans.
The tool sends messages and creates groups with delays to comply with Telegram limits.
